# Laravel Order Checkout

## Installation
- `git clone git@github.com:kbmulder1992/laravel-order-checkout.git`
- `cd docker-build`
- `./init.sh`

## Usages
The `docker-build` folder contains a `run.sh` script which will run the tests for the app
